# Suds 🍺

Suds is an Expo-based React Native mobile app for logging and tracking your drinks. It allows users to log drinks, attach photos, add locations via a map, and keep track of "Night Out" sessions.

## Features

- **Drink Logging**: Log the type of drink, name, quantity, location, and notes.
- **Photos**: Attach images to your drink logs from your camera roll.
- **Sessions**: Start a "Night Out" session to group multiple drinks under a single event.
- **Location Tracking**: Tag drinks with specific locations using maps.
- **Profiles**: View user profiles and track drink history over time.
- **Cross-Platform**: Built with React Native and Expo, with a web version available.

## Tech Stack

- **Framework**: [Expo](https://expo.dev) / React Native
- **Routing**: Expo Router
- **Styling**: [NativeWind](https://www.nativewind.dev/) (Tailwind CSS for React Native)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Data Fetching**: [TanStack React Query](https://tanstack.com/query/latest)
- **Backend / Database**: [Supabase](https://supabase.com/)
- **Forms**: React Hook Form

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the app**:
   ```bash
   npm run start
   ```

3. **Open on your device**:
   Scan the QR code with your Expo Go app (Android) or Camera app (iOS), or use `w` to open the web version.
